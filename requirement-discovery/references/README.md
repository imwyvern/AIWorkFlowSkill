### RICE 计算器

```
RICE = (Reach × Impact × Confidence) ÷ Effort

示例:
- Reach: 1000 用户
- Impact: 3 (高)
- Confidence: 80%
- Effort: 5 人天

RICE = (1000 × 3 × 0.8) ÷ 5 = 480
```
