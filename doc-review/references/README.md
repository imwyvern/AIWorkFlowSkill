## 评审检查清单

### 核心维度 (快速评审)

1. **完整性**
   - [ ] 背景目标清楚?
   - [ ] 功能清单完整?
   - [ ] 边界条件考虑?
   - [ ] 验收标准明确?

2. **一致性**
   - [ ] 术语统一?
   - [ ] 前后不矛盾?
   - [ ] 与现有系统不冲突?

3. **可行性**
   - [ ] 技术能实现?
   - [ ] 时间人力够?
   - [ ] 依赖稳定?

4. **安全性**
   - [ ] 接口有鉴权?
   - [ ] 无越权风险?
   - [ ] 敏感数据有保护?

5. **AI专项**
   - [ ] 模型选型合理?
   - [ ] Prompt设计清晰?
   - [ ] 成本可控?
   - [ ] 有降级方案?
   - [ ] Prompt注入有防护?
