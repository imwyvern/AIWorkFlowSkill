# Iteration Log

This file tracks each task iteration. Append entries, never overwrite.

Format:
```
## [YYYY-MM-DD HH:MM] Task: <brief description>
- **Changed**: what was modified
- **Verified**: build/lint/test results (pass/fail counts)
- **Issues found**: any failures or regressions
- **Next step**: what to do next (or "DONE" if complete)
- **Commit**: <hash> <message>
```
